# git gyakorlas szeruseg
## leírás a technológiákról
### git
* **Git** egy elosztott verziókezelő rendszer, amely lehetővé teszi a fejlesztők számára, hogy nyomon kövessék a kódváltozásokat, együttműködjenek másokkal, és kezeljék a projekt verzióit. A Git lehetővé teszi a fejlesztők számára, hogy helyi példányokat hozzanak létre a teljes kódtárból, így offline módban is dolgozhatnak, majd később szinkronizálhatják a változtatásokat a központi tárolóval.
### html
* **HTML** (HyperText Markup Language) egy jelölőnyelv, amelyet weboldalak és webalkalmazások struktúrájának meghatározására használnak. A HTML segítségével a fejlesztők meghatározhatják a weboldal elemeit, például címeket, bekezdéseket, képeket, linkeket és egyéb tartalmakat. A HTML dokumentumok böngészőkben jelennek meg, és alapvető építőkövei a webnek.

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Git Gyakorlas</title>
    <link rel="stylesheet" href="styles.css">
</head>
<body>
    <h1>Git Gyakorlas Szerűség</h1>
    <p>Ez egy példa HTML dokumentum a Git gyakorlas szerűséghez.</p>
    <script src="script.js"></script>
</body>
</html>
```
### css
* **CSS** (Cascading Style Sheets) egy stíluslap nyelv, amelyet a weboldalak megjelenésének és elrendezésének meghatározására használnak. A CSS segítségével a fejlesztők szabályokat hozhatnak létre a HTML elemek stílusára vonatkozóan, például színek, betűtípusok, margók és elrendezések beállítására. A CSS lehetővé teszi a weboldalak vizuális megjelenésének testreszabását és javítását.

```css
body {
    font-family: Arial, sans-serif;
    background-color: #f0f0f0;
    color: #333;
    margin: 20px;
}
```
### javascript
* **JavaScript** egy programozási nyelv, amelyet elsősorban weboldalak interaktív elemeinek létrehozására használnak. A JavaScript lehetővé teszi a fejlesztők számára, hogy dinamikus tartalmat hozzanak létre, kezeljék a felhasználói eseményeket, és kommunikáljanak a szerverrel. A JavaScript széles körben használt a webfejlesztésben, és számos könyvtár és keretrendszer (például React, Angular, Vue.js) épül rá, amelyek megkönnyítik a fejlesztési folyamatot.

```javascript
document.addEventListener('DOMContentLoaded', function() {
    console.log('Git Gyakorlas Szerűség oldal betöltve!');
});
```
